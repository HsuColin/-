# 「阿爸的家園」營養便當訂購系統 - 產品需求文件 (PRD)

## 1. 專案概述
本系統旨在為「阿爸的家園 (Healthy Nutrition Center)」打造一個直觀、專業且溫馨的線上訂餐平台。根據品牌理念「讓身體成為你想要的樣子」，系統將引導用戶根據自身營養需求（如增肌、孕期、銀髮保養等）選擇最適合的健康便當。

## 2. 目標用戶
*   **健身族群：** 尋求增肌減脂的高蛋白餐點。
*   **特殊營養需求者：** 孕婦、銀髮族、發育中兒童。
*   **健康管理需求者：** 需調整腸胃或進行運動營養補充的人士。
*   **在地居民與上班族：** 位於台北大同區，尋求高品質健康午餐的用戶。

## 3. 核心功能需求

### 3.1. 營養目標導航 (Goal-Oriented Navigation)
*   首頁提供六大營養目標入口：
    1. 增肌減脂
    2. 調整腸胃
    3. 運動營養補充
    4. 孕期營養
    5. 銀髮族保養
    6. 兒童健康長高營養
*   點擊後篩選出對應的推薦便當。

### 3.2. 便當展示與訂購
*   **產品詳情：** 包含名稱、主要營養成分、推薦理由。
*   **購物車：** 支持增加/減少數量。
*   **體驗價邏輯：** 系統自動判斷是否為首次訂購，提供 $49 體驗價優惠（每人限一次）。

### 3.3. 訂單結帳流程
*   **取餐資訊：** 選擇取餐日期與時間（限週一至週五 07:30-11:30）。
*   **聯絡資訊：** 姓名、電話。
*   **訂單確認：** 顯示總金額與取餐地點（台北市大同區承德路一段23號1樓）。

### 3.4. 品牌資訊與聯絡
*   整合地圖導航功能。
*   一鍵撥打電話（0906-000-923 / 02-25236643）。
*   營業時間與預約說明。

## 4. UI/UX 設計方針 (基於 Frontend Design Recipe 6)
*   **風格：** 溫暖有機、專業可信。
*   **色調：** 
    *   背景：溫暖米白色 (#f5f5f0)
    *   主色：健康綠 (#5A5A40)
    *   點綴色：活力橘 (對應傳單主題)
*   **字體：** 標題使用優雅的 Serif 字體，內文使用易讀的 Sans-serif。
*   **互動：** 使用 `motion` 實現平滑的頁面切換與按鈕反饋。

## 5. 技術架構
*   **前端：** React 19, TypeScript
*   **樣式：** Tailwind CSS 4
*   **動畫：** Motion
*   **圖標：** Lucide React
